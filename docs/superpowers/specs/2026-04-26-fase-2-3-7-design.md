### Mudanças em `tile_widget.dart`

Na camada de watermark (layer 1 do `Stack`):

```dart
// ANTES
Image.asset(
  animal.assetPath,
  opacity: const AlwaysStoppedAnimation(0.27),
  fit: BoxFit.contain,
  errorBuilder: (_, __, ___) => const SizedBox.shrink(),
)

// DEPOIS
Padding(
  padding: EdgeInsets.all(size * 0.08),
  child: SvgPicture.asset(
    animal.assetPath,
    fit: BoxFit.contain,
    colorFilter: null, // SVG colorido como produzido
    placeholderBuilder: (_) => const SizedBox.shrink(),
  ),
)
```

A opacidade `0.27` não é suportada diretamente por `SvgPicture.asset`. Aplicar via `Opacity` widget:

```dart
Padding(
  padding: EdgeInsets.all(size * 0.08),
  child: Opacity(
    opacity: 0.27,
    child: SvgPicture.asset(
      animal.assetPath,
      fit: BoxFit.contain,
      placeholderBuilder: (_) => const SizedBox.shrink(),
    ),
  ),
)
```

### Critério de aceite

- Galeria (item E) mostra watermark visível em todos os 11 tiles
- Número Fredoka Bold legível por cima em todos os tiles
- `HostArtwork` renderiza a imagem correta do animal em todos os 11
