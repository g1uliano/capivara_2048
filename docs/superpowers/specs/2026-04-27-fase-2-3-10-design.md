## Objetivo

Corrigir dois problemas de layout identificados em uso real após a 2.3.9:
1. `PauseButtonTile` grudado visualmente ao `StatusPanel` (recorde/pontuação/timer)
2. Anfitrião 1×1 pequeno demais — expandir pra 2×2

---

## Layout final

```
Padding(horizontal: 12dp)  ← padding pai compartilhado com BoardWidget
└── Column
    ├── GameHeader
    │   └── Column
    │       ├── LivesIndicator                    ← Linha A
    │       ├── SizedBox(height: 12)
    │       ├── Row                               ← Linha B+C
    │       │   ├── Expanded(flex:2)              ← cols 1-2
    │       │   │   └── HostBanner 2×2
    │       │   └── Expanded(flex:2)              ← cols 3-4
    │       │       └── Column
    │       │           ├── StatusPanel (152×72dp)
    │       │           ├── SizedBox(height: 6)
    │       │           └── Align(left) PauseButtonTile (72×72 em slot 152×72)
    │       └── SizedBox(height: 6)
    ├── BoardWidget
    ├── Spacer (ou SizedBox)
    └── InventoryBar
```

O `Row` da Linha B+C usa `crossAxisAlignment: CrossAxisAlignment.center` — garante que o bloco direito (StatusPanel + PauseButtonTile, ~150dp) fica centralizado verticalmente em relação ao anfitrião (~180dp), sobrando ~15dp em cada extremidade.

**Dimensões-chave** (com `tileSize=72`, `tileSpacing=8`):
- `slotWidth (2×1 ou 2×2)` = `2 × 72 + 8 = 152dp` (= `GameConstants.twoCellWidth` já existente)
- Altura do anfitrião 2×2 = `2 × 72 + 8 = 152dp` (imagem) + nome ~24dp + SizedBox(4) ≈ 180dp
- Altura do bloco direito = StatusPanel ~72dp + 6dp + PauseButtonTile 72dp = ~150dp
- `PauseButtonTile`: quadrado 72×72, `Align(Alignment.centerLeft)` dentro do slot 2×1

**Spacing vertical:**
- LivesIndicator → Row: 12dp
- StatusPanel → PauseButtonTile: 6dp
- GameHeader → BoardWidget: 6dp (via `SizedBox` trailing do GameHeader)

## Decisões de design

| Ponto | Decisão | Razão |
|-------|---------|-------|
| Posição do anfitrião | Esquerda (cols 1-2) | Preferência visual do usuário |
| Posição do StatusPanel + Pause | Direita (cols 3-4) | Complementar ao anfitrião |
| Col 3 standalone | Vazia | StatusPanel e Pause ocupam 2×1 (cols 3-4 juntas) |
| Alinhamento vertical do Pause | Centro da Linha C | Mais equilibrado visualmente |
| PauseButtonTile visual | 72×72 alinhado à esquerda no slot 2×1 | Aparência idêntica ao atual |
| BoxFit do anfitrião | `BoxFit.cover` | Todos os PNGs são quadrados — sem risco de corte |
| Placeholder vazio | Silhueta Capivara (opacity ~0.15) + "Comece!" sobreposto | Preenche espaço + sinaliza objetivo |
| Nome do anfitrião | Fredoka SemiBold 16sp | Aproveita largura dobrada |
| StatusPanel fontes | cronômetro 18sp, pontuação 24sp, recorde 13sp | Escala proporcional com espaço disponível |
| StatusPanel alinhamento | `CrossAxisAlignment.center` | Preenche slot 2×1 simetricamente |
| Spacing hierárquico | 12dp (Linha A→B+C), 6dp (StatusPanel→Pause, GameHeader→Board) | Separa info/controle |
| Alinhamento header/board | Padding pai compartilhado (12dp) | Alinhamento estrutural pixel-perfect |
| Galeria debug | Atualizar pra mostrar 2×2 na 2.3.10 | Validação visual dos 11 animais |
