- `Container` com `borderRadius: 12`

- Transições **negativas** (`current` diminui): fade apenas
- Identificação: widget recebe `int current` e `int previousCurrent`; se `current > previousCurrent` → positiva

### `PauseButtonTile`

**Tamanho:** tile-sized, calculado via `LayoutBuilder`.  
Fórmula: `tileSize = (boardWidth - 3 * gap) / 4`.  
O widget recebe `double tileSize` como parâmetro (calculado no `GameScreen` onde o `LayoutBuilder` já existe para o tabuleiro).

**Visual:**
```
Container(
  width: tileSize, height: tileSize,
  decoration: BoxDecoration(
    color: Colors.white,               // #FFFFFF
    borderRadius: BorderRadius.circular(12),
    border: Border.all(color: Color(0xFFFF8C42), width: 3),
    boxShadow: [BoxShadow(offset: Offset(0,2), blurRadius: 4, color: Colors.black.withOpacity(0.25))],
  ),
  child: Column(
    mainAxisAlignment: MainAxisAlignment.center,
    children: [
      Icon(Icons.pause_circle_filled, color: Color(0xFFFF8C42), size: tileSize * 0.50),
      SizedBox(height: 4),
      OutlinedText("Pausar", style: Fredoka SemiBold 12sp, fillColor: white, outlineColor: black, strokeWidth: 1),
    ],
  ),
)
```

**Animação de tap:** `GestureDetector` com `onTapDown`/`onTapUp` controlando `AnimationController` → scale 1.0→0.95→1.0 em 100ms total.

**`onTap`:** chama `notifier.pause()`.

**Visual quando pausado:** sem mudança — o botão permanece como "Pausar" sempre; retomar é feito pelo `PauseOverlay`.

**Semântica:** `Semantics(label: 'Botão Pausar')`.
