# Fase 3.0 — E2E Harness & Smoke Tests Implementation Plan

> **REQUIRED SUB-SKILL:** Use the executing-plans skill to implement this plan task-by-task.

**Goal:** Construir a infraestrutura mínima (harness + scenario class + registry) e provar que funciona com 5 smoke tests passando, antes de adicionar o catálogo completo.

**Architecture:** `GameTestHarness` cria `ProviderContainer` com Hive em diretório temp + SharedPreferences mock, expõe `boot()`/`restart()`/`teardown()`. Cada cenário é uma instância de `E2EScenario` (id, title, tags, run-function). Um único entry point `test/e2e/run_all_test.dart` itera por `allScenarios` e gera um `testWidgets` por cenário.

**Tech Stack:** Flutter test, Riverpod, Hive (path_provider já existe), shared_preferences (mock), nenhuma dependência nova.

**Design referência:** `docs/plans/2026-05-04-e2e-test-framework-design.md`

```dart
// test/e2e/_harness/scenario.dart
import 'package:flutter_test/flutter_test.dart';
import 'test_harness.dart';

enum ScenarioTag {
  critical,
  slow,
  demo,
  tier1_only,
  tier2_only,
  golden,
}

typedef ScenarioRun = Future<void> Function(
  WidgetTester tester,
  GameTestHarness harness,
);

class E2EScenario {
  final String id;
  final String title;
  final Set<ScenarioTag> tags;
  final ScenarioRun run;

  const E2EScenario({
    required this.id,
    required this.title,
    required this.tags,
    required this.run,
  });

  bool matchesTag(ScenarioTag tag) => tags.contains(tag);
}
```

```dart
// test/e2e/_harness/test_harness.dart
import 'dart:io';
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'package:hive/hive.dart';
import 'package:shared_preferences/shared_preferences.dart';
import 'package:capivara_2048/app.dart';
import 'package:capivara_2048/data/models/daily_rewards_state.dart';
import 'package:capivara_2048/data/models/daily_rewards_state_adapter.dart';
import 'package:capivara_2048/data/models/game_state.dart';
import 'package:capivara_2048/data/models/inventory.dart';
import 'package:capivara_2048/data/models/inventory_hive_adapter.dart';
import 'package:capivara_2048/data/models/lives_state.dart';
import 'package:capivara_2048/data/models/lives_state_adapter.dart';
import 'package:capivara_2048/data/models/personal_records.dart';
import 'package:capivara_2048/data/models/personal_records_hive_adapter.dart';
import 'package:capivara_2048/data/models/game_record_hive_adapter.dart';
import 'package:capivara_2048/data/repositories/game_record_repository.dart';
import 'package:capivara_2048/core/providers/reduce_effects_provider.dart';
import 'package:capivara_2048/domain/inventory/inventory_notifier.dart';
import 'package:capivara_2048/domain/lives/lives_notifier.dart';
import 'package:capivara_2048/domain/daily_rewards/daily_rewards_notifier.dart';
import 'package:capivara_2048/presentation/controllers/game_notifier.dart';
import 'package:capivara_2048/presentation/controllers/personal_records_notifier.dart';
import 'package:capivara_2048/presentation/controllers/settings_notifier.dart';

class GameTestHarness {
  late ProviderContainer container;
  late Directory tempDir;
  bool _booted = false;

  Future<Widget> boot({
    GameState? initialGameState,
    Inventory? initialInventory,
    PersonalRecords? initialRecords,
    DailyRewardsState? initialDaily,
    LivesState? initialLives,
  }) async {
    SharedPreferences.setMockInitialValues({});
    tempDir = await Directory.systemTemp.createTemp('e2e_hive_');
    Hive.init(tempDir.path);
    _registerAdapters();

    final prefs = await SharedPreferences.getInstance();
    final repo = GameRecordRepository();
    await repo.load();

    container = ProviderContainer(
      overrides: [
        settingsProvider.overrideWith((ref) => SettingsNotifier(prefs)),
        gameRecordRepositoryProvider.overrideWithValue(repo),
        if (initialGameState != null)
          gameProvider.overrideWith(
            (ref) => GameNotifier(ref.read(gameEngineProvider), ref)
              ..debugSetState(initialGameState),
          ),
      ],
    );

    await container.read(reduceEffectsProvider.notifier).load();
    await container.read(inventoryProvider.notifier).load();
    await container.read(dailyRewardsProvider.notifier).load();
    await container.read(personalRecordsProvider.notifier).load();

    if (initialInventory != null) {
      container.read(inventoryProvider.notifier).debugSetState(initialInventory);
    }
    if (initialRecords != null) {
      container.read(personalRecordsProvider.notifier).debugSetState(initialRecords);
    }
    if (initialDaily != null) {
      container.read(dailyRewardsProvider.notifier).debugSetState(initialDaily);
    }
    if (initialLives != null) {
      container.read(livesProvider.notifier).debugSetState(initialLives);
    }

    _booted = true;
    return UncontrolledProviderScope(
      container: container,
      child: const CapivaraApp(),
    );
  }

  Future<void> teardown() async {
    if (_booted) {
      container.dispose();
      await Hive.close();
      if (tempDir.existsSync()) await tempDir.delete(recursive: true);
      _booted = false;
    }
  }

  void _registerAdapters() {
    if (!Hive.isAdapterRegistered(LivesState.hiveTypeId)) {
      Hive.registerAdapter(LivesStateAdapter());
    }
    if (!Hive.isAdapterRegistered(Inventory.hiveTypeId)) {
      Hive.registerAdapter(InventoryHiveAdapter());
    }
    if (!Hive.isAdapterRegistered(DailyRewardsState.hiveTypeId)) {
      Hive.registerAdapter(DailyRewardsStateAdapter());
    }
    if (!Hive.isAdapterRegistered(PersonalRecords.hiveTypeId)) {
      Hive.registerAdapter(PersonalRecordsHiveAdapter());
    }
    if (!Hive.isAdapterRegistered(GameRecordHiveAdapter.kTypeId)) {
      Hive.registerAdapter(GameRecordHiveAdapter());
    }
  }
}
```

> ⚠️ Atenção aos imports: alguns notifiers podem não ter `debugSetState` exposto. Se compilar quebrar, verifique e adicione um `debugSetState` à classe (sob `@visibleForTesting`) ou use o método público equivalente. Lista de classes a verificar: `LivesNotifier`, `InventoryNotifier`, `DailyRewardsNotifier`, `PersonalRecordsNotifier`. Se tiver que adicionar, faça PR atômico antes de continuar.

```dart
// test/e2e/_harness/tester_extensions.dart
import 'package:flutter/material.dart';
import 'package:flutter_test/flutter_test.dart';

enum SwipeDirection { up, down, left, right }

extension E2EHelpers on WidgetTester {
  Future<void> tapByKey(String key) async {
    await tap(find.byKey(Key(key)));
    await pumpAndSettle();
  }

  Future<void> swipeBoard(SwipeDirection dir) async {
    // Procura BoardWidget pelo Key estável; se não existe, usa center da tela.
    final boardFinder = find.byKey(const Key('game_board'));
    final target = boardFinder.evaluate().isNotEmpty
        ? getCenter(boardFinder)
        : Offset(view.physicalSize.width / 2, view.physicalSize.height / 2);

    const distance = 200.0;
    final delta = switch (dir) {
      SwipeDirection.up => const Offset(0, -distance),
      SwipeDirection.down => const Offset(0, distance),
      SwipeDirection.left => const Offset(-distance, 0),
      SwipeDirection.right => const Offset(distance, 0),
    };

    await dragFrom(target, delta);
    await pumpAndSettle();
  }
}
```

```dart
// dentro de GameTestHarness
Future<Widget> restart() async {
  if (!_booted) throw StateError('cannot restart before boot');
  container.dispose();
  // NÃO apagar tempDir nem Hive.close — boxes persistem.
  // Re-cria container com mesmo tempDir.
  final prefs = await SharedPreferences.getInstance();
  final repo = GameRecordRepository();
  await repo.load();

  container = ProviderContainer(
    overrides: [
      settingsProvider.overrideWith((ref) => SettingsNotifier(prefs)),
      gameRecordRepositoryProvider.overrideWithValue(repo),
    ],
  );
  await container.read(reduceEffectsProvider.notifier).load();
  await container.read(inventoryProvider.notifier).load();
  await container.read(dailyRewardsProvider.notifier).load();
  await container.read(personalRecordsProvider.notifier).load();

  return UncontrolledProviderScope(
    container: container,
    child: const CapivaraApp(),
  );
}
```
