## Contexto técnico

### Padrões de boot estabelecidos
```dart
// HomeScreen
Future<void> _bootToHome(WidgetTester tester, GameTestHarness harness) async {
  final widget = await tester.runAsync(() => harness.boot());
  await tester.pumpWidget(widget!);
  await tester.pumpAndSettle(const Duration(seconds: 5));
}

// GameScreen
Future<void> _bootToGame(WidgetTester tester, GameTestHarness harness) async {
  final widget = await tester.runAsync(() => harness.boot());
  await tester.pumpWidget(widget!);
  await tester.pumpAndSettle(const Duration(seconds: 5));
  await tester.gotoGame(harness);
}
```

### Pump strategy consolidada
- Após state changes: `pump()` + `pump(const Duration(milliseconds: 500))`
- `pumpAndSettle(Duration(seconds: 5))` só para splash/boot
- Após operações async Hive: `tester.runAsync(() async { await Future.delayed(Duration(milliseconds: 200)); })`

### Providers relevantes
```dart
// settings (SharedPreferences)
settingsProvider → SettingsNotifier → SettingsState { hapticEnabled, locale }
// setHaptic(bool) / setLocale(String) — persiste em SharedPreferences

// reduce effects (SharedPreferences)
reduceEffectsProvider → ReduceEffectsNotifier → bool
// toggle() — persiste em SharedPreferences

// personalRecords (Hive)
personalRecordsProvider → PersonalRecordsNotifier → PersonalRecords { highestLevelEver }

// gameRecords (Hive)
gameRecordRepositoryProvider → GameRecordRepository (in-memory + Hive)
// .all returns List<GameRecord>

// daily rewards (Hive)
dailyRewardsProvider → DailyRewardsNotifier → DailyRewardsState { currentDay, lastClaimedDate, claimedThisCycle }
// .claim(DateTime.now()) — avança dia, salva no Hive
// applyClaim: currentDay < 7 → nextDay = currentDay + 1

// lives (Hive)
livesProvider → LivesNotifier → LivesState { lives, regenCap, earnedCap, lastRegenAt }
```

### Pause button
Sem Key estável no PauseButtonTile. Usar:
```dart
// Opção A: Semantics label
await tester.tap(find.bySemanticsLabel('Botão Pausar'));

// Opção B: via notifier (mais confiável em headless)
harness.container.read(gameProvider.notifier).pause();
```

### Game timer
`GameState.elapsedMs` incrementa a cada 100ms via `Timer.periodic` quando jogo está rodando.
`isPaused: true` → timer para (chama `_stopTimer()`).

### Restart
`gameProvider.notifier.restart()` → novo jogo, score=0, board novo.

## Ajustes esperados durante execução

1. **`persistence.high_score_survives_restart`** — `GameState.highScore` é in-memory, não persiste separadamente entre sessões. O implementador deve adaptar o cenário para testar o que realmente persiste (ex: após `confirmGameOver`, o high score fica nos game records). Alternativa: simplificar para verificar que o score reseta após restart (comportamento esperado documentado).

2. **`pause.system_back_keeps_paused_state`** — `navigator.pop()` direto pode não funcionar se não houver rota a desempilhar. Verificar se é necessário usar `tester.pageBack()` ou se chamar `Navigator.of(context).pop()` via notifier.resume() + pop é mais correto.

3. **`pause.game_doesnt_consume_time_while_paused`** — `tester.pump(Duration)` avança o clock do fake timer, mas o `Timer.periodic` do game engine pode ou não ter sido iniciado. Se o timer não foi iniciado (nenhum move feito), `elapsedMs` sempre será 0. A versão acima faz 1 swipe antes para iniciar o timer.

4. **`daily.cycle_resets_after_streak_break`** — verificar `applyStreakReset`: `currentDay` reseta para 1, depois `applyClaim` avança para 2. O cenário espera `currentDay == 2`.

5. **`settings.reduce_effects_disables_blur_in_pause`** — `BackdropFilter` é renderizado condicionalmente em `PauseOverlay`. Com `reduceEffects=true`, o código usa `Container(color: Color(0xE6000000))` em vez de `BackdropFilter`. Verificar no código de `PauseOverlay` se `BackdropFilter` está dentro de `if (!reduceEffects)`.

6. **Imports `personalRecordsProvider`** — importar de `lib/presentation/controllers/personal_records_notifier.dart`

7. **Import `gameRecordRepositoryProvider`** — importar de `lib/data/repositories/game_record_repository.dart`

8. **Import `BackdropFilter`** — é do Flutter SDK: `import 'dart:ui'; BackdropFilter` (já disponível via `flutter/material.dart`)
